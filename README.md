# Line Follower Car

Ce projet ROS 2 (Jazzy) simule une voiture autonome dans Gazebo Sim capable de suivre une ligne blanche au sol . La detection de la ligne est realisee via une camera embarquee et un traitement d images avec OpenCV.

## Lancer le projet
```bash
ros2 launch line_follower_car line_follower.launch.py
```
